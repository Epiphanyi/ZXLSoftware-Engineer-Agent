# 贪吃蛇游戏 (Snake Game)

一个使用Python和Pygame库实现的经典贪吃蛇游戏。

## 功能特点

- 经典的贪吃蛇游戏玩法
- 分数系统（每吃一个食物得10分）
- 蛇的长度显示
- 游戏暂停/继续功能
- 游戏结束检测（撞墙或撞到自己）
- 重新开始游戏功能
- 网格背景显示
- 详细的控制说明

## 运行要求

- Python 3.6+
- Pygame 2.5.2

## 安装和运行

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行游戏：
   ```bash
   python main.py
   ```

## 游戏控制

- **方向键 (↑ ↓ ← →)**: 控制蛇的移动方向
- **P键**: 暂停/继续游戏
- **空格键**: 游戏结束后重新开始
- **ESC键**: 退出游戏

## 游戏规则

1. 控制蛇吃掉红色食物
2. 每吃一个食物，蛇的长度增加，得分增加10分
3. 避免蛇撞到墙壁或自己的身体
4. 游戏结束后可以按空格键重新开始

## 开发说明

游戏使用面向对象编程设计：
- `SnakeGame` 类：处理游戏循环、显示和用户输入
- `Game` 类：处理游戏逻辑（蛇的移动、碰撞检测、食物生成）

网格系统：
- 屏幕大小：800×600像素
- 网格大小：20×20像素
- 网格数量：40×30个网格
- 游戏速度：10帧/秒
